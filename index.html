Tables

Table Elements

Tag Name Description
table Table This element is the baseline for creating a table.
thead Table Heading A way to section off the table heading.
tbody Table Body A way to section off long tables.
tfoot Table Footer A way to section off the table footer.
th Table Heading Defines data as a heading.
tr Table Row This element defines what is in a table row.
td Table Data Container for information in a row.

Table Attributes

Attribute Name Arguments Description
scope Scope col, row, colgroup, rowgroup Determines if th is used for headings for a single column or row, or if it is used for headings that sit over the top of multiple columns or rows.
colspan Column Span Unitless number ie. 2, 5, etc. Determines how many columnns a td should span.
rowspan Row Span Unitless number ie. 2, 5, etc. Determines how many columnns a td should span.


CSS: The Box Model

The Box Model comprises the set of properties that define parts of an element that take up space on a web page. The model includes the content area’s size (width and height) and the element’s padding, border, and margin.

Margin

The amount of space between the border and the outside edge of the element

Border

The thickness and style of the border surrounding the content area and padding. There are two main CSS properties that affect the border:

  • border-width is set in pixels or with one of the following key words
    • thin
    • medium
    • thick
  • border-style is set with any of the following key words
    • none
    • hidden
    • dotted
    • dashed
    • solid
    • double
    • groove
    • ridge
    • inset
    • outset
    • a mix of different borders

Padding

The amount of space between the content area and the border. You can adjust specific padding by adding -top, -right, -left, or -bottom. Or, you can simply input 4 pixel values and it will adjust the padding clockwise, starting with top.

  • 3 values
    1. top
    2. LR
    3. bottom
  • 2 values
    1. top/bottom
    2. LR

Content

The space where your content is shown. Defined by the width and height properties in CSS.

Box-Sizing

You can avoid dimensional issues with resizing and readjusting by using border-box. In this model, the height and width of the box will remain fixed. Border and padding are included inside of the box, which means overall dimensions of the box do not change. The default value of box-sizing is content-box.

Overflow

The overflow property sets the desired behavior when content does not fit in the parent element box (overflows) in the horizontal and/or vertical direction. It is defined by a few key values.

overflow: visible overflow: hidden overflow: clip overflow: scroll overflow: auto

CSS Selectors

Selector Name Description
* { } Universal Selector References the entire CSS unless otherwise noted.
.class { } Class Selector References any class from the HTML.
#id { } ID Selector References any element ID from the HTML.
h1.special Element Class Selector Selects elements with a specific class. In this example, only h1 elements with the special class are selected.
:hover On Hover Something will happen to the element, class, or id before :hover when the mouse is hovering over it.
For more selectors, visit W3schools' CSS Selector Reference sheet.

A few other important notes about selectors. #id are the most specific selectors in CSS and they should only be applied to a single element. <style> allows you to write CSS in an HTML file.
